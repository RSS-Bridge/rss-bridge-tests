Nach Meta-Einsteig: Google zieht sich offenbar bei Scale AI als Kunde zurück

2025-06-15 12:23:00

by: Niklas Jan Engelking

Diese Woche stieg der Meta-Konzern mit 49 Prozent bei Scale AI ein. Das hat anscheinend auch Auswirkungen auf die restliche Scale-Kundschaft.

(Bild: Shutterstock)

Google will einem Medienbericht zufolge seine Ausgaben bei Scale AI massiv kürzen. Beim zweiteren Unternehmen erwarb diese Woche der Meta-Konzern 49 Prozent der Anteile – für insgesamt 14,1 Milliarden US-Dollar. Scale AI fungiert als Dienstleister, um KI-Modelle für bestimmte Aufgaben zu trainieren. Zu den Kunden gehören einerseits Tech-Unternehmen, aber zum Beispiel auch Industriekonzerne oder Regierungen.

Eigentlich wollte Google dieses Jahr 200 Millionen US-Dollar als Kunde von Scale AI ausgeben, berichtet die Nachrichtenagentur Reuters. Wie die Journalisten aber aus informierten Kreisen erfahren haben wollen, führe der Tech-Konzern nun Gespräche mit Wettbewerbern von Scale, um die Kundenbeziehungen zu Scale zu beenden.

Ähnliche Pläne offenbar auch bei Microsoft

Demnach gibt es bei Microsoft ähnliche Überlegungen, OpenAI soll eine entsprechende Entscheidung schon vor mehreren Monaten getroffen haben. Allerdings soll der KI-Hersteller weiterhin in geringerem Umfang mit Scale AI zusammenarbeiten.

Entwickler von generativen KI-Modellen sind auch die größten Kunden von Scale AI, weil sie den Zugang zu Spezialisten benötigen, welche die Trainingsdaten für ihre Modelle aufbereiten können. Zudem spielen Dienstleister wie Scale für die Entwickler selbstfahrender Autos eine wichtige Rolle. Auch die US-Regierung gehört zu Scales Kunden und wird das laut Reuter-Informationen auch weiterhin sein.

Google selbst wollte die Entwicklung gegenüber Reuters nicht kommentieren, ebenso wenig Scale AI. Ein Scale-Sprecher betonte allerdings, dass das Geschäft weiterhin gut laufe und Scale AI auch weiter als unabhängiges Unternehmen arbeiten würde.

Zuvor hatten sich in dieser Woche die Gerüchte bestätigt, dass Meta 14,3 Milliarden US-Dollar in Scale AI investiert, um im KI-Rennen weiter ganz vorn mitzuspielen. Alexandr Wang, Gründer und CEO von Scale, wird Berichten zufolge direkt Mark Zuckerberg unterstellt, bleibt aber im Aufsichtsgremium von Scale AI. Er soll ein neues KI-Team bei Meta leiten. Dadurch, dass Meta nur 49 Prozent von Scale AI übernimmt, sichert sich Meta auf der einen Seite die KI-Expertise des Unternehmens, umgeht aber wettbewerbsrechtliche Auflagen. Dennoch ist es möglich, dass die Wettbewerbsbehörde der USA sich den Deal genauer anschauen wird.

    Angreifer können Server über Schwachstelle in Dell iDRAC Tools attackieren

    2025-06-15 11:40:00

    by: Dennis Schirrmacher

    Eine Schwachstelle in Dells Servermanagementsoftware iDRAC Tools gefährdet Server. Eine reparierte Version steht zum Download.

    (Bild: AFANASEV IVAN/Shutterstock.com)

    Angreifer können an einer Sicherheitslücke in Dell iDRAC Tools ansetzen, um Server zu attackieren. Mittlerweile haben die Entwickler die Schwachstelle geschlossen.

    Sicherheitsproblem

    Die Lücke (CVE-2025-27689) ist mit dem Bedrohungsgrad "hoch" versehen. Um eine Attacke einleiten zu können, muss ein lokaler Angreifer bereits über niedrige Nutzerrechte verfügen. Ist das gegeben, kann er sich auf einem nicht näher beschriebenen Weg höhere Rechte verschaffen.

    Die Entwickler versichern, die Schwachstelle in iDRAC Tools 11.3.0.0 geschlossen zu haben. Alle vorigen Ausgaben sind verwundbar. Sie weisen in einem Beitrag darauf hin, dass der Sicherheitspatch nur in Kombination mit bestimmten Windows-Server-2025-Versionen Systeme effektiv schützt.

    • V51N7 - Windows Server 2025 Standard
    • JT4MH - Windows Server 2025 Datacenter
    • G5F67 - Windows Server 2025 Essentials

    Der Einfachheit halber bietet Dell im Zuge seines Dell-Digital-Locker-Angebots ein Bundle bestehend aus der gepatchten iDRAC-Version inklusive einer passenden Windows-Server-2025-Ausgabe zum Download.

    • Sicherheitslücken Tenable Agent: Angreifer können Systemdaten überschreiben

    2025-06-15 10:54:00

    by: Dennis Schirrmacher

    Die Entwickler haben in der aktuellen Ausgabe von Tenable Agent mehrere Sicherheitslücken geschlossen.

    (Bild: Alfa Photo/Shutterstock.com)

    Der Schwachstellenscanner Tenable Agent ist derzeit selbst verwundbar. Angreifer können an drei Schwachstellen ansetzen, um Systeme zu attackieren. Bislang gibt es keine Berichte zu laufenden Attacken. Eine reparierte Version steht zum Download bereit.

    Sicherheitsupdate installieren

    In einer Warnmeldung führen die Entwickler aus, dass sie insgesamt drei Lücken (CVE-2025-36631 "hoch", CVE-2025-36632 "hoch", CVE-2025-36633 "hoch") geschlossen haben. Den Beschreibungen der Schwachstellen zufolge können nicht-administrative Benutzer Attacken ausführen. Klappt das, können Angreifer unter anderem Dateien mit Systemrechten manipulieren oder sogar löschen. Außerdem kann Schadcode auf PCs gelangen.

    Wie solche Attacken im Detail ablaufen könnten, ist bislang unklar. Unbekannt ist derzeit auch, woran Admins bereits attackierte Computer erkennen können. Die Entwickler geben an, die Schwachstellen in Tenable Agent 10.8.5 geschlossen zu haben. Alle vorigen Versionen sollen angreifbar sein.
